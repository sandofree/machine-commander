# machine-commander
[MachineCommander](https://www.zhgcloud.com/) is an IoT system to manage all the construction machines working on construction projects.

## MCP server config
```json
{
    "mcpServers": {
        "MachineCommander": {
            "command": "uvx",
            "args": ["machinecommander"]
        }
    }
}
```
